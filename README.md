# 为什么整这个项目？

工作中经常会学习一些新技术，通常情况下，没有实践过，过一段时间就会忘记，徒增学习成本。

dk 本人很喜欢看小说，so，开了这个工程，将遇到的感兴趣的技术、框架都怼一遍实践项目。古代将军练兵讲究以战养兵，进步神速，dk 效仿之。
